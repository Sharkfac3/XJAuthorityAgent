# Writing: Callout Conventions

## Era callouts
Use these exact formats whenever a procedure or spec applies to only one era. Never assume — always callout.

```
🔴 Era 1 — Renix (1988–1990):
[Era 1 specific content]

🟡 Era 2 — OBD-I (1991–1995):
[Era 2 specific content]

🔵 Era 3 — OBD-II (1996–2001):
[Era 3 specific content]

⚡ 1996 Transition Year:
[1996-specific content]
```

## Transmission callouts
```
🔁 Automatic Only (AW4):
[Content that applies only to AW4-equipped vehicles]

⚙️ Manual Transmission:
[Content that applies only to manual-equipped vehicles]
```

## Safety callouts
```
⚠️ WARNING:
[Safety-critical information — injury or vehicle damage risk]

🔌 DISCONNECT BATTERY before proceeding.
[Use before any procedure involving live circuits]
```

## Trail reliability callouts
```
🏔️ Trail Note:
[Information specifically relevant to off-road reliability or trail use]
```

## Tip callouts
```
💡 Tip:
[Helpful information that is not required but improves the outcome]
```

## Notes on usage
- Do not overuse callouts — if everything is called out, nothing stands out
- A chapter with more than 4–5 callout boxes should be reviewed for over-formatting
- Era callouts are mandatory whenever a spec, procedure, or part number differs between eras
- Safety callouts are mandatory before any step involving electrical disconnection, fuel system work, or hot components
- Trail reliability callouts are used when a choice has different implications for street vs trail use
