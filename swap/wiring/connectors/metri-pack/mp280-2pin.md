# Metri-Pack 280: 2-Pin

## Used on
- O2 sensor signal wires — Era 2 and Era 3
- Note: O2 sensor heater circuit uses a separate 2-wire connector (blade terminals), not Metri-Pack

## Physical description
- Series: Metri-Pack 280 (larger terminal than MP150)
- Pin count: 2 (signal circuit only — heater is separate)
- Terminal size: 2.8mm — visibly larger than MP150
- Release tool: Metri-Pack 280 terminal release pick (different from MP150 tool)

## Terminal
- Part number: Delphi 12010799 (female, 18 AWG) — verify current availability
- Wire range: 18 AWG

## Pinout
| Pin | Circuit |
|---|---|
| A | O2 signal (0–1V) |
| B | O2 signal ground |

## Notes
- Do not use MP150 release tool on MP280 terminals — wrong tool will damage the cavity
- O2 sensor pigtails are available pre-made — easier than terminal replacement in most cases
- Heater circuit (2-wire) is a separate connector using blade terminals — not covered here
