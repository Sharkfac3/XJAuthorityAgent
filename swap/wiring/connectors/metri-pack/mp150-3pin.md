# Metri-Pack 150: 3-Pin

## Used on
- TPS (throttle position sensor) — Era 2 and Era 3
- MAP sensor — Era 2 and Era 3
- IAC valve — Era 2 and Era 3
- ⚡ 1996: Same connector as Era 2 for all three

## Physical description
- Series: Metri-Pack 150
- Pin count: 3
- Body color: Varies by application
- Locking: Secondary lock tab
- Wire seal: Individual cavity seals

## Terminal
- Part number: Delphi 12010973 (female, 18–20 AWG)
- Wire range: 18–20 AWG
- Release tool: Metri-Pack 150 terminal release pick

## Pinout — TPS
| Pin | Circuit |
|---|---|
| A | TPS ground |
| B | TPS signal (0.5V–4.5V) |
| C | 5V reference |

## Pinout — MAP
| Pin | Circuit |
|---|---|
| A | MAP ground |
| B | MAP signal (0.5V–4.5V) |
| C | 5V reference |

## Pinout — IAC (4-wire stepper uses two 2-pin connectors on some years — verify)
| Pin | Circuit |
|---|---|
| A | Coil A+ |
| B | Coil A− |
| C | Coil B+ (if 3-pin used) |

## Important note
TPS, MAP, and IAC connectors are physically identical — the 3-pin MP150 body is the same across all three applications. **Label every connector during removal.** Swapping the TPS and MAP connectors will cause immediate no-start or severe driveability problems.
