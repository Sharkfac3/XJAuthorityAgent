# Axles Interchange Index

## Purpose
Interchange decisions for donor axles targeting the Jeep Cherokee XJ — fitment, dimensional compatibility, gearing, and driveshaft specifications.

## Scope
Axle and differential donor interchange guidance. This branch owns the "will it fit and what changes" questions. For donor vehicle identification (what to pull, junkyard tips), see `sourcing/donor-vehicles/axles/`. For install procedures, see `work/swaps/axles/`.

## What belongs here
- donor-to-XJ dimensional comparisons
- fitment-decision notes (narrowing, steering, brakes, width)
- gearing and driveshaft compatibility
- part-family comparisons across donor classes

## What does not belong here
- install procedures → `work/swaps/axles/`
- stock-only fact baselines → `vehicle/systems/axles/`
- symptom-first troubleshooting → `diagnostics/`
- donor vehicle identification and junkyard tips → `sourcing/donor-vehicles/axles/`
