# Era 1: Oxygen Sensor (O2)

## Specs
- Type: Narrowband zirconia, heated
- Quantity: 1 (upstream only, no downstream)
- Output: 0–1V (rich = 0.7–0.9V, lean = 0.1–0.3V, stoich crossover ~0.45V)
- Heater: 12V supply, ~10–15 ohm resistance
- Location: Exhaust manifold or downpipe
- Warm-up time: ~30–60 seconds before signal is valid

## Testing
At warm idle in closed loop, signal should oscillate between ~0.1V and ~0.9V approximately once per second. Flat signal = failed sensor or open circuit.

## Swap notes
- Narrowband O2 is retained for closed-loop fuel correction at idle and cruise
- For tuning, add a wideband O2 sensor — see `work/swaps/ecu/sensors-and-inputs/wideband-o2.md`
- Standard 4-wire heated O2 connector — compatible with most aftermarket sensors
