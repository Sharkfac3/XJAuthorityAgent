# Era 2: Oxygen Sensor (O2)

## Specs
- Type: Narrowband zirconia, heated
- Quantity: 1 upstream only — no downstream sensor on OBD-I
- Output: 0–1V (rich = 0.7–0.9V, lean = 0.1–0.3V)
- Heater: 12V supply
- Connector: Standard 4-wire heated O2 — Metri-Pack 280 2-pin signal + 2-wire heater
- Location: Exhaust manifold or downpipe

## Testing
At warm idle in closed loop, signal oscillates ~0.1–0.9V approximately once per second. Flat signal = failed sensor, open circuit, or sensor not yet at operating temperature.

## Swap notes
- Narrowband O2 retained for closed-loop correction at idle and cruise
- Add a wideband O2 for tuning — see `swap/ecu-selection/wideband.md`
- Standard 4-wire connector — any compatible aftermarket sensor fits
