# 2WD Vs 4WD

## Purpose
Provide the stock baseline differences that change XJ answers when drivetrain configuration is the main split.

## Scope
Use this file before answering questions about transfer cases, front-axle hardware, driveshaft layout, underbody packaging, and drivetrain-related service or diagnosis.

## Core difference
A 2WD XJ and a 4WD XJ do not share the same stock driveline layout.
The most important first-hop difference is simple:
- **2WD** has no transfer case and no driven front axle in normal stock form
- **4WD** adds a transfer case, front driveline hardware, and front-axle involvement that materially changes parts, diagnosis, and procedures

## What changes with 4WD
On a stock 4WD XJ, expect the answer to depend on:
- transfer-case family and engagement hardware
- front driveshaft presence
- front axle and related u-joint / shaft service context
- drivetrain vibration paths that do not exist on 2WD in the same way
- shifter, linkage, and floor / tunnel packaging tied to the transfer case

## What does not change just because the Jeep is 4WD
4WD status does **not** replace the need to confirm:
- year or era
- engine family
- transmission family
- ABS status when braking or axle hardware is the actual split

## High-value expert-agent uses
Use this file when the user is really asking:
- whether transfer-case hardware should exist on the vehicle
- whether a front driveline complaint is even possible in stock form
- whether a service procedure or sourcing answer must branch by drivetrain layout first
- whether a suspension, axle, or brake answer may differ because the Jeep is 4WD

## Current limits
Current canon does not yet split out a full stock 2WD-vs-4WD hardware matrix by year, axle family, or package.
Use this file as the first configuration filter, then route into the owning stock, work, diagnostics, or sourcing branch.
