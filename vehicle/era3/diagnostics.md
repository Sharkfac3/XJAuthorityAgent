# Era 3: Diagnostics

## Reading codes
Connect any generic OBD-II scanner to the 16-pin DLC (under dash, driver's side).
No key cycling needed — scanner reads live and stored codes directly.

## Common OBD-II codes on Era 3 XJs
| Code | Description | Common XJ cause |
|---|---|---|
| P0107 | MAP low voltage | Failed MAP, cracked vacuum hose |
| P0108 | MAP high voltage | MAP wiring short |
| P0117 | CTS low voltage | Shorted CTS |
| P0118 | CTS high voltage | Open CTS circuit |
| P0121 | TPS range/performance | Worn TPS, intermittent wiper |
| P0122 | TPS low voltage | TPS ground or sensor |
| P0123 | TPS high voltage | TPS reference wiring |
| P0131 | O2 low (upstream) | Failed upstream O2 |
| P0134 | O2 no activity (upstream) | Sensor cold, failed, or open |
| P0136 | O2 low (downstream) | Failed downstream O2 |
| P0300 | Random misfire | Multiple causes — see below |
| P0301–P0306 | Cylinder misfire | Injector, plug, wire, compression |
| P0335 | CPS no signal | CPS failure, wiring, connector |
| P0340 | Cam sensor no signal | Cam sensor failure |
| P0441 | EVAP incorrect purge | Purge solenoid, hose, canister |
| P0455 | EVAP large leak | Loose gas cap, cracked hose |
| P1297 | No change in MAP | MAP sensor stuck |
| P1391 | Intermittent CPS or cam loss | Connector corrosion, sensor failing |

## Pre-swap checklist
- [ ] Scan and record all stored codes
- [ ] Clear codes, drive cycle, rescan — distinguish current from history codes
- [ ] CPS signal verified (no P0335)
- [ ] Cam sensor verified (no P0340)
- [ ] TPS sweep smooth (no P0121/P0122/P0123)
- [ ] MAP reads barometric at key-on engine-off
- [ ] Both O2 sensors functional (upstream oscillating, downstream present)
- [ ] No misfire codes (P0300–P0306)
- [ ] Fuel pressure at 49 psi key-on (39 psi on 1996)
- [ ] EVAP codes noted — not critical for swap but document before proceeding
- [ ] All sensor connectors inspected — no corrosion, broken locks
- [ ] Ground straps clean and tight
