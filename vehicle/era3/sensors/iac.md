# Era 3: Idle Air Control (IAC) Valve

## Specs
Identical to Era 2. See `vehicle/era2/sensors/iac.md` for full specs.
4-wire stepper motor, ~50–55 ohm per coil.

## Connector
- 1996: Era 2 connector — see `vehicle/era3/1996-notes.md`
- 1997–2001: Metri-Pack 150 — same as Era 2

## Swap notes
Same stepper motor type across all three eras. Same open source ECU support considerations apply.
See `swap/ecu-selection/iac-strategy.md` for options if ECU lacks native stepper support.
