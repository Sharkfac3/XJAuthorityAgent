# Era 3: Oxygen Sensors (O2)

## Upstream sensor (pre-cat)
Identical to Era 2 upstream O2. See `vehicle/era2/sensors/o2.md` for full specs.
Used for closed-loop fuel correction — retained and wired to open source ECU O2 input.

## Downstream sensor (post-cat)
- Added on all Era 3 vehicles for catalyst efficiency monitoring
- Type: Narrowband, heated — same output range as upstream (0–1V)
- Location: After catalytic converter
- Purpose: OBD-II catalyst monitor only — not used for fuel control
- Connector: Standard 4-wire heated O2

## Swap notes
- Upstream O2: Wire to open source ECU narrowband O2 input — required for closed-loop operation
- Downstream O2: Not required for engine management — can be left disconnected at ECU
- Disconnecting downstream O2 will set P0136/P0141 codes and disable catalyst readiness monitor
- If emissions testing is required, downstream O2 must remain connected and functional
- If open source ECU has a spare analog input, downstream O2 can be connected for logging purposes
