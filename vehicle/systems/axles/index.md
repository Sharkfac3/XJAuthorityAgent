# Axles Index

## Purpose
Route stock axle content for the Jeep Cherokee XJ.

## Use this branch when
The main question is about factory axle configuration, identification, specifications, gear ratios, or differential options.

## First-hop loading path

- Axle family overview and year matrix → [overview.md](overview.md)
- Axle identification (by year, visual, tag) → [identification.md](identification.md)
- Stock axle specifications → [stock-specs.md](stock-specs.md)
- Gear ratio identification and selection → [gear-ratio-identification.md](gear-ratio-identification.md)
- Locker and limited-slip options → [locker-options.md](locker-options.md)

## Individual axle detail leaves

- Dana 30 front → [dana30-front.md](dana30-front.md)
- Dana 35 rear → [dana35-rear.md](dana35-rear.md)
- Chrysler 8.25 rear → [chrysler-825-rear.md](chrysler-825-rear.md)

## What belongs here
- stock axle architecture and component families
- identification guides
- specification tables
- factory and aftermarket differential options
- year/package/trim axle splits

## What does not belong here
- service or installation procedures → `work/`
- symptom-first diagnostics → `diagnostics/`
- donor-part sourcing or interchange → `sourcing/`
- swap execution → `work/swaps/axles/`

- Vehicle systems hub → [`../index.md`](../index.md)
