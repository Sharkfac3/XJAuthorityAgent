# Stock XJ Axle Specifications

Tabular reference for every factory axle installed in the 1984–2001 Jeep Cherokee XJ.
Use this file as a dimensional baseline when comparing donor axles for swap projects.

---

## Year-range axle summary

| Year range | Front axle | Rear axle | Notes |
|------------|-----------|-----------|-------|
| 1984–1986 | Dana 30 (260 U-joints) | Dana 35 (27-spline) | No C8.25 option |
| 1987–1989 | Dana 30 (260 U-joints) | Dana 35 (27-spline) | Dana 44 rear with Tow Package |
| 1990 | Dana 30 (260 U-joints) | Dana 35 (27-spline) | — |
| 1991–1994 | Dana 30 (260 U-joints) | Dana 35 or C8.25 (27-spline) | C8.25 on non-ABS models |
| 1995–1996 | Dana 30 (297 U-joints) | Dana 35 or C8.25 (27-spline) | C8.25 on non-ABS models |
| 1997–2001 | Dana 30 (297 U-joints) | Chrysler 8.25 (29-spline) | Standard across all trims |

For identification procedures, see [identification.md](identification.md).

## Front Axle — Dana 30

All XJ model years used the Dana 30 front axle (4WD models only).

| Spec | Value |
|------|-------|
| Type | Open-knuckle, reverse-cut ring gear |
| Housing width (WMS-to-WMS) | ~57.2" |
| Axle tube OD | 2.5" |
| Axle shaft spline count | 27-spline (all years) |
| U-joint size (inner) | 260 (84–94), 297 (95–01 / ABS-equipped) |
| U-joint size (outer / stub) | 260 |
| Ring gear diameter | 7.125" |
| Carrier bolts | 10 |
| Axle bolt pattern | 5 on 4.5" |
| Differential | Open (standard), Trac-Lok (optional), vacuum disconnect (some years) |
| Common gear ratios | 3.07, 3.55, 3.73, 4.10 (I-4/V6) |
| Spring type | Leaf spring, spring-under-axle |
| Spring pad width | ~2.5" |
| Spring pad spacing (center-to-center) | ~32" |
| Pinion rotation | Reverse (passenger-side ring gear) |
| Approximate weight | ~180 lb assembled |

### Year notes
- 1984–1994: 260-size inner U-joints, no ABS
- 1995–2001: 297-size inner U-joints on ABS-equipped models — stronger, recommended upgrade for earlier years
- Vacuum disconnect front axle on some 2WD/4WD part-time models (eliminates when aftermarket shafts are installed)

## Rear Axle — Dana 35

Standard rear axle for most XJ model years.

| Spec | Value |
|------|-------|
| Type | Semi-float |
| Housing width (WMS-to-WMS) | ~58.5" |
| Axle tube OD | 2.5" |
| Axle shaft spline count | 27-spline |
| U-joint size (at diff) | N/A (one-piece shafts, C-clip retained) |
| Ring gear diameter | 7.5625" |
| Carrier bolts | 10 |
| Axle bolt pattern | 5 on 4.5" |
| Differential | Open (standard), Trac-Lok (optional) |
| Retention | C-clip |
| Common gear ratios | 3.07, 3.55, 3.73, 4.10 (I-4/V6) |
| Spring type | Leaf spring, spring-under-axle |
| Spring pad width | ~2.5" |
| Spring pad spacing (center-to-center) | ~42" |
| Approximate weight | ~150 lb assembled |

### Known weakness
The Dana 35 is the weakest rear axle offered in the XJ. C-clip retention means a broken shaft can allow the wheel to separate from the vehicle. Not recommended for tires larger than 31" or aggressive off-road use without an axle shaft upgrade or full axle swap.

## Rear Axle — Chrysler 8.25

Available 1991–2001. Standard on 1997–2001 XJs.

| Spec | Value |
|------|-------|
| Type | Semi-float |
| Housing width (WMS-to-WMS) | ~58.5" |
| Axle tube OD | 3.0" |
| Axle shaft spline count | 27-spline (91–96), 29-spline (97–01) |
| Ring gear diameter | 8.25" |
| Carrier bolts | 10 |
| Axle bolt pattern | 5 on 4.5" |
| Differential | Open (standard), Trac-Lok (optional) |
| Retention | Bolt-in (no C-clips) |
| Common gear ratios | 3.07, 3.55, 3.73 |
| Spring type | Leaf spring, spring-under-axle |
| Spring pad width | ~2.5" |
| Spring pad spacing (center-to-center) | ~42" |
| Approximate weight | ~170 lb assembled |

### Advantage over Dana 35
Bolt-in axle shafts (no C-clips), larger ring gear, and 29-spline shafts on 97+ models. Strength is comparable to a Dana 44. The 8.25 is the best stock rear axle commonly found in XJs.

## Rear Axle — Dana 44 (Tow Package)

Available only on 1987–1989 XJs equipped with the factory Towing Package. Rare.

| Spec | Value |
|------|-------|
| Type | Semi-float |
| Housing width (WMS-to-WMS) | ~58.5" |
| Axle tube OD | 3.125" |
| Axle shaft spline count | 30-spline |
| Ring gear diameter | 8.5" |
| Carrier bolts | 10 |
| Axle bolt pattern | 5 on 4.5" |
| Differential | Open or Trac-Lok |
| Retention | Bolt-in (no C-clips) |
| Common gear ratios | 3.07, 3.55 |
| Spring type | Leaf spring, spring-under-axle |
| Spring pad width | ~2.5" |
| Spring pad spacing (center-to-center) | ~42" |
| Approximate weight | ~190 lb assembled |

### Notes
Strongest factory rear axle in any XJ. Direct bolt-in to all XJ years. Extremely difficult to find — most XJ owners never encounter one at a junkyard.
