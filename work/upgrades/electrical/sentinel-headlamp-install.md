# Sentinel Headlamp Feature Installation

## Purpose
Enable the Sentinel headlamp delay feature on XJ and MJ vehicles — allows headlights to stay on briefly after the ignition is turned off, providing lighting when exiting at night.

## How it works
A headlamp delay module relay plugs into an existing connector already present in the vehicle. No custom wiring is required — the factory harness was pre-wired for this feature; it simply was not included on all trim levels.

---

## Parts

| Part | Details |
|------|---------|
| Headlamp delay module relay | Part number **56000636** or **56009324** |
| Cost | Approximately $50 |

---

## Installation
Plug the relay into the existing pre-wired connector in the vehicle's electrical system. No wiring modifications needed.

---

## Notes
- Vehicles upgraded with aftermarket HID headlights may experience an issue where the Sentinel feature does not allow the lights to shut off properly. An upgraded headlight wiring harness resolves this incompatibility.
- See `work/upgrades/electrical/headlight-harness-upgrade.md` for the relay harness upgrade if HID lights are installed.
