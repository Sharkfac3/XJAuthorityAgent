# Repairs Index

## Purpose
Route XJ repairs work.

## Scope
Use this branch for fixing failed or damaged components while preserving the same general system role.

## What belongs here
- repairs workflows
- branch-specific project or procedure folders
- compatibility notes that only matter in the context of this work class
- validation or follow-up stubs when they are specific to this work class

## What does not belong here
- stock baseline facts with no action context
- symptom-first troubleshooting trees
- buying-only comparisons with no work context
- agent instructions or repo governance
