# Inspections Index

## Purpose
Route XJ inspections work.

## Scope
Use this branch for condition assessment, pre-purchase review, and validation checks.

## What belongs here
- inspections workflows
- branch-specific project or procedure folders
- compatibility notes that only matter in the context of this work class
- validation or follow-up stubs when they are specific to this work class

## What does not belong here
- stock baseline facts with no action context
- symptom-first troubleshooting trees
- buying-only comparisons with no work context
- agent instructions or repo governance

## Child branches
- [`baseline-condition/index.md`](baseline-condition/index.md) — Condition assessment for a stock or unknown-condition XJ.
- [`pre-purchase/index.md`](pre-purchase/index.md) — Inspection workflows for evaluating a vehicle before purchase.
- [`post-work-validation/index.md`](post-work-validation/index.md) — Validation checks after service, repair, upgrade, or swap work.
