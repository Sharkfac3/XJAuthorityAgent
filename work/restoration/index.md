# Restoration Index

## Purpose
Route XJ restoration work.

## Scope
Use this branch for stock-correct, preservation-oriented, or refurbishment work.

## What belongs here
- restoration workflows
- branch-specific project or procedure folders
- compatibility notes that only matter in the context of this work class
- validation or follow-up stubs when they are specific to this work class

## What does not belong here
- stock baseline facts with no action context
- symptom-first troubleshooting trees
- buying-only comparisons with no work context
- agent instructions or repo governance

## Child branches
- [`body/index.md`](body/index.md) — Stock-oriented body restoration, preservation, and factory-correct exterior work.
- [`interior/index.md`](interior/index.md) — Stock-oriented interior restoration and trim preservation work.
- [`electrical/index.md`](electrical/index.md) — Restoration-oriented repair of factory electrical systems and original harness condition.
- [`wiring/index.md`](wiring/index.md) — Harness restoration, connector rehabilitation, and stock wiring preservation work.
- [`trim/index.md`](trim/index.md) — Badge, molding, fastener, and trim restoration topics.
- [`weather-sealing/index.md`](weather-sealing/index.md) — Doors, hatch, glass, and seal restoration topics.
