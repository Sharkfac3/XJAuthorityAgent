# Setup And Calibration Index

## Purpose
Route XJ and MJ setup, adjustment, and calibration work.

## Use this branch when
The main question is how to set something correctly after installation, service, repair, or modification.

## First-hop loading path
- Engine setup and adjustment → [`engine/index.md`](engine/index.md)
- Drivability calibration → [`drivability/index.md`](drivability/index.md)
- Alignment and geometry setup → [`alignment/index.md`](alignment/index.md)
- Brake setup and bedding → [`brakes/index.md`](brakes/index.md)
- ECU configuration and tuning setup after engine-management changes → [`ecu/index.md`](ecu/index.md)

## What belongs here
- setup and calibration workflows
- adjustment and baseline-setting tasks
- post-install validation that is mainly about settings, not fault isolation
- narrow cross-links to stock, diagnostic, and sourcing branches when needed

## What does not belong here
- stock baseline facts with no action context
- symptom-first troubleshooting trees
- buying-only comparisons with no work context
- agent instructions or repo governance
