# Setup And Calibration Index

## Purpose
Route XJ setup and calibration work.

## Scope
Use this branch for adjustment, setup, and calibration tasks.

## What belongs here
- setup and calibration workflows
- branch-specific project or procedure folders
- compatibility notes that only matter in the context of this work class
- validation or follow-up stubs when they are specific to this work class

## What does not belong here
- stock baseline facts with no action context
- symptom-first troubleshooting trees
- buying-only comparisons with no work context
- agent instructions or repo governance

## Child branches
- [`engine/index.md`](engine/index.md) — Adjustment and setup tasks for stock or modified engine operation.
- [`drivability/index.md`](drivability/index.md) — Calibration and setup tasks tied to idle, throttle response, and running quality.
- [`alignment/index.md`](alignment/index.md) — Alignment and geometry setup workflows.
- [`brakes/index.md`](brakes/index.md) — Brake adjustment, bedding, and setup workflows.
- [`ecu/index.md`](ecu/index.md) — Engine-management setup and calibration topics during transition from legacy swap content.
