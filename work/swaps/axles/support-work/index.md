# One-Ton Axle Swap — Support Work

## Purpose
Route cross-cutting support work that applies to both axles or the vehicle as a whole during a one-ton axle swap.

## Use this branch when
- the question is about driveshaft measurement, fabrication, or installation
- the question is about ring-and-pinion setup or locker installation
- the question is about lift height planning, suspension travel, or fender clearance
- the topic spans both the front and rear axle swaps

## Do not start here when
- the question is specific to the front axle → [`../one-ton-front/index.md`](../one-ton-front/index.md)
- the question is specific to the rear axle → [`../one-ton-rear/index.md`](../one-ton-rear/index.md)
- the question is about donor selection → [`../../../sourcing/donor-vehicles/axles/index.md`](../../../sourcing/donor-vehicles/axles/index.md)
- the question is about fitment dimensions → [`../../../sourcing/interchange/axles/index.md`](../../../sourcing/interchange/axles/index.md)
