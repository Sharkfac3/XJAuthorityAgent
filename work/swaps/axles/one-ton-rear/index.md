# One-Ton Rear Axle Swap — Sterling 10.50

## Purpose
Route the rear axle swap workflow for installing a 2005+ Super Duty Sterling 10.50 under a Jeep Cherokee XJ.

## Use this branch when
- the question is about preparing, narrowing, or modifying the Sterling 10.50 rear housing
- the question is about welding spring perches and setting pinion angle on the rear axle
- the question is about rear disc brake integration or e-brake cable routing
- the question is about rear shock mounts, bump stops, traction bars, or SOA mounting

## Do not start here when
- the question is about which donor to target → [`../../../sourcing/donor-vehicles/axles/ford-super-duty-sterling-10-50.md`](../../../sourcing/donor-vehicles/axles/ford-super-duty-sterling-10-50.md)
- the question is about fitment dimensions or width decisions → [`../../../sourcing/interchange/axles/super-duty-sterling-fitment.md`](../../../sourcing/interchange/axles/super-duty-sterling-fitment.md)
- the question is about the front axle → [`../one-ton-front/index.md`](../one-ton-front/index.md)
- the question is about gearing or driveshafts → [`../support-work/index.md`](../support-work/index.md)

## Procedure sequence

Work through these leaves in order:

1. **Housing preparation** — bracket removal, narrowing, axle shaft shortening
   → [`housing-prep.md`](housing-prep.md)

2. **Spring perch welding** — perch placement, pinion angle setting, weld procedure
   → [`spring-perch-welding.md`](spring-perch-welding.md)

3. **Brakes and e-brake** — disc brake retention, e-brake drum-in-hat integration, cable routing, proportioning
   → [`brake-and-ebrake.md`](brake-and-ebrake.md)

4. **Mounting and brackets** — SOA configuration, shock mounts, bump stops, traction bars, u-bolt plates
   → [`mounting-and-brackets.md`](mounting-and-brackets.md)
