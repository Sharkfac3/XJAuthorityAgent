# Packard 56: CPS Connector

## Used on
- Crankshaft position sensor (CPS) — Era 2 and Era 3

## ⚠️ Two connector styles — different pigtails
| Years | Style | Notes |
|---|---|---|
| 1991–1992 | Round-pin (Packard 56 series) | 3-pin round body |
| 1993–2001 | Flat-pin (Packard Weather Pack / Metri-Pack 630) | Flat rectangular body |

Confirm which style is present before ordering a replacement pigtail. The sensor itself is the same; only the connector pigtail differs.

## Pinout (both styles)
| Pin | Circuit |
|---|---|
| A | CPS signal + |
| B | CPS signal − |
| C | Shield / ground |

## Notes
- CPS is a VR (Variable Reluctance) sensor — polarity matters
- Reversing A and B pins shifts the timing reference — engine may not start or will run with incorrect timing
- Mark wire orientation before disconnecting
- Connector corrosion is the most common CPS-related failure on 1991–1992 round-pin style

## Release procedure
- Round-pin (1991–92): Depress locking tab, pull straight off
- Flat-pin (1993+): Depress secondary lock, pull straight off
- Terminal release tool: Packard 56 pick for round; Metri-Pack 630 pick for flat
