# Metri-Pack: Overview

## Identity
- Family: Delphi (now Aptiv) Metri-Pack
- Used on: All sensor connections on Era 2 and Era 3 XJs
- Not used on Era 1 (Renix uses its own proprietary connectors)

## Series in use on XJ
| Series | Terminal size | Primary XJ application |
|---|---|---|
| 150 | 1.5mm | TPS, MAP, CTS, IAT, IAC |
| 280 | 2.8mm | O2 sensor signal wires |
| 630 | 6.3mm | Higher-current circuits |

## Key characteristic
Each series has a different terminal size, release tool, and seal. Using the wrong tool will damage terminals. Confirm the series before attempting terminal removal.

## General release procedure
1. Identify the series by connector body size and terminal cavity size
2. Use the correct Metri-Pack terminal release tool for the series
3. Insert tool into the terminal cavity from the wire side
4. Depress the secondary lock (if present) first
5. Push terminal forward while withdrawing wire rearward
6. Never pull the wire without releasing the terminal lock — terminal will deform

## Sourcing
- Connector bodies, terminals, and seals: Available from Delphi/Aptiv distributors, Waytek Wire, Del City
- Pre-made pigtails: Available from XJ specialty suppliers and general automotive wiring suppliers
- Terminal release tools: Available as a set — buy the full Metri-Pack tool kit, not individual picks
