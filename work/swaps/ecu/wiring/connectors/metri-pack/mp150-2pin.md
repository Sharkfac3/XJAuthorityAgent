# Metri-Pack 150: 2-Pin

## Used on
- CTS (coolant temperature sensor) — Era 2 and Era 3
- IAT (intake air temperature sensor) — Era 2 and Era 3
- ⚡ 1996: Same connector as Era 2

## Physical description
- Series: Metri-Pack 150
- Pin count: 2
- Body color: Varies (gray, black, natural)
- Locking: Secondary lock tab on connector body
- Wire seal: Individual cavity seals, one per wire

## Terminal
- Part number: Delphi 12010973 (female, 18–20 AWG) — verify current availability
- Wire range: 18–20 AWG
- Release tool: Metri-Pack 150 terminal release pick

## Pinout
| Pin | Circuit |
|---|---|
| A | Sensor signal (NTC return to ECU) |
| B | Sensor ground (5V reference on some configurations — verify) |

## Notes
- CTS and IAT use identical 2-pin MP150 connectors — label both sensors during removal
- Connector body is keyed — will only fit in one orientation
- If terminal is corroded, replace the terminal — do not sand or file the contact surface
