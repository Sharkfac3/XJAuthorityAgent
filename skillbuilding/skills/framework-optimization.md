# Skill: Framework Optimization

Use this skill when the task is to improve `skillbuilding/` itself rather than canonical repo content.

## Steps
1. Read `AGENTS.md`.
2. Read `skillbuilding/index.md`.
3. Read `skillbuilding/agent-design-rules.md`, `skillbuilding/skill-design-rules.md`, `skillbuilding/operating-modes.md`, `skillbuilding/artifact-contracts.md`, and `skillbuilding/meta-improvement-loop.md`.
4. Read `skillbuilding/agents/framework-architect.md`.
5. Read only the specific `skillbuilding/` files involved in the friction being addressed.
6. Classify the framework problem.
7. Choose the lightest fix that improves composability, consistency, routing, or lifecycle coverage.
8. Return a concrete improvement plan or apply the approved framework edits.

## Expected outputs
- framework problem statement
- problem class
- affected files
- recommended fix size
- exact file changes or additions
- expected benefit to the framework

## Success condition
The result should make `skillbuilding/` more consistent, more composable, or easier to extend without adding unnecessary structure.
