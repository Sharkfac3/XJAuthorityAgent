# Skill: Topic Expansion

Use this skill when one topic is known to be weak and needs a structured plan to become authoritative.

## Steps
1. Read `AGENTS.md`.
2. Read `skillbuilding/index.md`.
3. Read `skillbuilding/gap-taxonomy.md` and `skillbuilding/evidence-standards.md`.
4. Read `skillbuilding/agents/research-planner.md`.
5. Read the current canonical file for the topic, if one exists.
6. Build a topic brief using `skillbuilding/templates/topic-brief.md`.
7. Identify the smallest set of follow-on tasks needed to close the gap.

## Expected outputs
- current state
- missing facts/procedures
- blocked decisions
- target canonical files
- validation needs
- recommended work order

## Use cases
- AW4 integration expansion
- 1996 exception mapping
- injector data hardening
- trigger setup clarification
- ECU platform comparison upgrades
